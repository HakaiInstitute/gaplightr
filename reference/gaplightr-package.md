# gaplightr: Gap Light Analysis of Virtual and Real Fisheye Photography

Analyzes forest canopy gap light transmission using LiDAR point cloud
data and hemispherical photography. Provides tools for processing LiDAR
data, computing horizon angles, and calculating gap light metrics.

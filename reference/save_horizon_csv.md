# Save horizon data to CSV file

Save horizon data to CSV file

## Usage

``` r
save_horizon_csv(horizon_df, point_id, output_dir)
```

## Arguments

- horizon_df:

  Data frame with horizon data (azimuth, horizon_height, x_msk, y_msk)

- point_id:

  Integer unique point identifier

- output_dir:

  Directory to save CSV file

## Value

File path where CSV was saved

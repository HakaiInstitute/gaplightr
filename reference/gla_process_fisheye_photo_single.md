# Process single fisheye photo and calculate SSR

Process single fisheye photo and calculate SSR

## Usage

``` r
gla_process_fisheye_photo_single(
  solar_data,
  img_file,
  lat_deg,
  long_deg,
  Kt,
  elev_res,
  azi_res,
  rotation_deg,
  keep_gap_fraction_data = FALSE,
  radial_distortion = "equidistant",
  threshold = 0
)
```
